# Titanic Survival Analysis

## 📌 Project Overview
This project analyzes the Titanic dataset using Python. The objective is to understand the factors that affected passenger survival through data cleaning, analysis, and visualization.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
- Titanic Dataset (train.csv)

## 📊 Analysis Performed
- Data Loading
- Data Cleaning
- Handling Missing Values
- Gender-wise Survival Analysis
- Passenger Class Survival Analysis
- Age Group Analysis
- Data Visualization

## 📈 Visualizations
- Survival by Gender
- Survival by Passenger Class
- Age Distribution
- Survival by Age Group

## 📌 Conclusion
- Female passengers had a higher survival rate than males.
- First-class passengers had better survival chances.
- Children had a higher survival rate than adults.
- Data visualization helped identify survival patterns effectively.
